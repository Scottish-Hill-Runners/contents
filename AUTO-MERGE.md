# Auto-merge label policy

1. Only SHR officials may apply the `auto-merge` label — never the PR author.
2. Eligible changes: editorial/content-only edits in non-sensitive files, max 5 files / 150 lines.
3. All status checks must be green and no merge conflicts before labelling.
4. Auto-merge targets `staging` only; `main` publishes are always manual.
5. Any official may remove the label at any time; workflow can be disabled if problems arise.
